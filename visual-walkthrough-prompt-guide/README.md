# Visual Walkthrough Prompt Guide

Use this prompt to generate slide decks for any technical topic using Gamma.

👉 **[Open Gamma](https://gamma.1stcollab.com/vishakha.sadhwani_ig)**

---

## Prompt

```
Create a step-by-step visual guide for [YOUR TOPIC] as a slide deck. Use a hand-drawn sketch style with pastel fills, black outlines, and a light cream background. Each slide should have a numbered step title, a one-line bold summary, a highlighted quote/tip, and an architecture-style diagram showing the flow. Keep diagrams icon-heavy with minimal text and rounded boxes connected by arrows.
```

Replace `[YOUR TOPIC]` with your subject — e.g. *deploying an LLM with vLLM and Docker*, *Kubernetes network policies*, *setting up an EC2 GPU instance*.
